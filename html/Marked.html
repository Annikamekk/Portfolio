Project info

In the first year of my CMD studies, I did a project related to internet standaarden, which required me to create a website with HTML and CSS. This was a mobile project, so we had to make it phone-only. The website is not responsive.

The intention of this project was to offer international students a digital guide on a specific topic of the persona you have chosen.

This was my first individual project, so everything was new to me, and on top of that I had hardly any experience in writing HTML and CSS. So it was definitely a challenge, but I thought it was a fun project and I'm proud of the outcome.

Client: Hogeschool van Amsterdam
